Compilation:
-----------
To compile all programs:

    make

To compile one file only:

    make Q6
    make Q7
    make Q8

Running:
--------
Run each program like this:

    ./Q6 <One argument is needed>
    ./Q7 <Two arguments are needed>
    ./Q8 <One argument is needed>

Example for programs needing threads:

    ./Q6 4
    ./Q7 3 10000 
    ./Q8 16

Cleaning:
---------
To remove all compiled files:

    make clean
